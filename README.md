## Cloud Drive (Next.js)


## Getting Started

1. Clone the repository

```
git clone https://github.com/ziaq/cloud-drive
```

2. Navigate to the project directory
```
cd cloud-drive
```

3. Install dependencies

```
npm i
```

4. Build

```
npm run build
```

5. Start the app

```
npm start
```

The app will be available at: [http://localhost:3000](http://localhost:3000 "http://localhost:3000")
